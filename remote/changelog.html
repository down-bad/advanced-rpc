1.5.1 - December 10, 2022
  • Fixed {year} variable
  • New animated artworks should appear now without having to clear cache
  • Resolved an error
All requested animated artworks have already been added, keep them coming! With this update, the artworks file should properly be fetched every time you open Cider. If you want to keep your animated artworks up to date without restarting Cider, simply open this changelog modal and the latest version of the artworks file will be fetched.
1.5.0 - December 6, 2022
  • Animated Artwork support! However, not all albums are supported yet. If you come across an album with an animated artwork that is not supported by AdvancedRPC, please use the "Request animated artwork" option from the sidebar to request it, and it will be added as soon as possible.
  • Added the following text variables: trackCount, genre, year, albumId, artistId
  • Added the following URL variables: albumUrl, artistUrl, youtubeUrl, youtubeMusicUrl
  • Podcasts: Added applePodcastsUrl and websiteUrl for buttons. appleMusicUrl will no longer work so be sure to replace it
  • Back-end changes, presence should be more accurate
  • Fix library songs missing artwork
  • Artwork support for iCloud songs has been disabled as the Cider API for it is dead
  • Resolved errors
  • Updated fonts
