1.2.0 - August 4, 2022
Added an advanced settings section with the following new settings:
  • Delay presence info updating in order to avoid Discord rate limits, resulting in the presence freezing
  • Set a custom image when the artwork doesn't exist or can't be loaded
  • Change the size of the artwork
  • Choose when to apply AdvancedRPC settings (manually, on state change or immediately)
Other changes include:
  • Ability to put artwork as small image
  • Show changelog when there is a new update available
  • Show notification on startup when there is a new update available
  • Added footer displaying version info and credits
  • Small design tweaks
  • Other tweaks and back-end changes
