1.4.0 - October 28, 2022
  • Ability to customize your presence for Videos, Radio Stations & Podcasts, with new variables for each content type
  • New sidebar for easy access to each configuration
  • Show artwork for user-uploaded/iCloud songs
  • Added Spotify URL variable for buttons
  • Improved the logic and responsiveness of your presence
  • Ability to automatically remove potentially invalid buttons
  • Fixed presence not showing after installing the plugin and not changing any settings
  • Ability to use text variables in URL fields
  • Added song ID as variable
  • Removed presence delay option
  • Removed text field variables (mirroring a text field using a variable)
  • Fixed Apple Music URLs for videos
  • Fixed "Apply Changes" bar sometimes appearing when no changes have been made
  • Improved fonts
  • Lots of UI & animation improvements, back-end changes and more

Please make sure to report any issues or request new features by opening an issue on GitHub. Thank you for using AdvancedRPC!
