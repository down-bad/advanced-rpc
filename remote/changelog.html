1.5.0 - December 6, 2022
  • Animated Artwork support! However, not all albums are supported yet. If you come across an album with an animated artwork that is not supported by AdvancedRPC, please use the "Request animated artwork" option from the sidebar to request it, and it will be added as soon as possible.
  • Added the following text variables: trackCount, genre, year, albumId, artistId
  • Added the following URL variables: albumUrl, artistUrl, youtubeUrl, youtubeMusicUrl
  • Podcasts: Added applePodcastsUrl and websiteUrl for buttons. appleMusicUrl will no longer work so be sure to replace it
  • Back-end changes, presence should be more accurate
  • Fix library songs missing artwork
  • Artwork support for iCloud songs has been disabled as the Cider API for it is dead
  • Resolved errors
  • Updated fonts
1.4.1 - November 3, 2022
Improvements and bug fixes
1.4.0 - October 28, 2022
  • Ability to customize your presence for Videos, Radio Stations & Podcasts, with new variables for each content type
  • New sidebar for easy access to each configuration
  • Show artwork for user-uploaded/iCloud songs
  • Added Spotify URL variable for buttons
  • Improved the logic and responsiveness of your presence
  • Ability to automatically remove potentially invalid buttons
  • Ability to use text variables in URL fields
  • Added song ID as variable
  • Fixed Apple Music URLs for videos
  • Fixed presence not showing after installing the plugin and not changing any settings
  • Fixed "Apply Changes" bar sometimes appearing when no changes have been made
  • Removed presence delay option
  • Removed text field mirroring using variables (variables generated from text fields)
  • Improved fonts
  • Lots of UI & animation improvements, back-end changes and more

Please make sure to report any issues or request new features by opening an issue on GitHub. Thank you for using AdvancedRPC!
